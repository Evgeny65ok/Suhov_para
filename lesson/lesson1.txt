так чтобы произведения чисел на каждой из трех линий
были одинаковыми. Какое число окажется 
в закрашенном кружке?

2)Решалось алгоритмами на C++
3)Вся математика проходит в отдельном файле math.cpp
4)В файле main.cpp работает прогамма, все вычисления в math.cpp
   
bool used[10]; 
void findSolution(int pos) {
    if (pos == 10) { 
        if (check()) {

        }
        return;
    }
    for (int i = 1; i <= 9; i++) {
        if (!used[i]) {
            used[i] = true;
            circles[pos] = i;
            findSolution(pos + 1);
            used[i] = false;
